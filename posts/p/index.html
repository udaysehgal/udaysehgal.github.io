Task: Installing Terraform & Working with AWS

Installing Terraform

We tried to install Terraform using these set of commands: (mentioned on the official website)

sudo apt-get update  
sudo apt-get install -y gnupg software-properties-common curl
curl -fsSL https://apt.releases.hashicorp.com/gpg | sudo apt-key add -
sudo apt-add-repository "deb [arch=amd64] https://apt.releases.hashicorp.com $(lsb_release -cs) main"
sudo apt-get update 
sudo apt-get install terraform

But, we were having an issue with the second command (apt-key). We moved towards manual installation as we had faced a similar type of error earlier as well and we couldn’t debug it.

For the manual installation, we simply downloaded the package & unzipped it. We need to change the location of the binary terraform. We use the following set of commands:

echo $PATH
mv ~/Downloads/terraform /usr/local/bin/

This moves the terraform binary to our required location.

Installing aws client

We follow these sets of commands to install the aws client

curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

After this, we need to enter our access keys on the aws cli. For that we first enter the command & then our access keys as well as secret access key.

aws configure

(The recording has not been uploaded for the above process for obvious purposes)

Our AWS cli is ready to use and we can start using it!
