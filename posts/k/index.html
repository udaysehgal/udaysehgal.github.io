Task for Today : Installing Ansible

Steps I followed for installing ansible:

    $ sudo apt update
    $ sudo apt install software-properties-common
    $ sudo add-apt-repository --yes --update ppa:ansible/ansible
    $ sudo apt install ansible

Ran into an error. Looked up the error online and tried to solve it using a command (https://askubuntu.com/questions/732985/force-update-from-unsigned-repository). Tried that below, but to no avail.
