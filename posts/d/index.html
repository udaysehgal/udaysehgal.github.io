I installed Hugo today and got around to building my static website. I was curious about what a static website actually is so I searched for the difference between a static website and a dynamic one. So I learnt that a static website is one in which the content is displayed to the viewer as it is stored in the user’s web browser. They do not take inputs from the viewer. I then understood that I had been creating static websites earlier as well, but I was just not aware that they were static websites. I have created a few websites in HTML,CSS,JS before this internship but I was just not aware that they were called static websites.

I initialised Hugo and downloaded all the files and set up a basic webpage in the time before I made more changes.
