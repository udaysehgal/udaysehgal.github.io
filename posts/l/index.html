Configuring SSH on Ansible

Now, we move on to to make it possible for our node to access the Ansible server. We do this via SSH. We generate a SSH key with the command:

$ ssh-keygen

A default name is kept for the file without a password. We display the contents of the public key with the command:

$ cat ~/.ssh/id_rsa.pub

Our next plan of action would be to copy the text thatâ€™s displayed and log into our node server. We issue the command:

$ sudo -s

Next, we need to open the authorized_keys file. The command used:

$ sudo nano ~/.ssh/authorized_keys

Paste the contents of the server key at the bottom of this file and save.
